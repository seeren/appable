#  Babel skeleton
## 🎉 Features
>  Babel skeleton for build es6 JavaScript in the browser
* [Es6](https://www.npmjs.com/package/babel-cli)  watch
* [Sass](https://www.npmjs.com/package/node-sass)  watch

## 📦 Installation
Install package
```
npm install babel-skeleton
```
## 🚀 Serve
Run the application
```
npm run serve
```
## ⛏ Test
Pass the tests
```
npm run test
```
Generate report
```
npm run coverage
```
